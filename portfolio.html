reddish

Reddish is a React app that uses the Reddit API to search for topics, browse through subreddits, and explore Reddit content. Users can view posts, read comments, and navigate different communities seamlessly. The app is built with Redux Toolkit for efficient state management and was developed collaboratively in a team of three. The design focuses on a clean and minimalistic interface, making it easy to discover and engage with discussions across Reddit.

The idea behind building the app was to deepen our understanding of Reddit‚Äôs structure and functionality while improving our skills in working with external APIs. Initially, integrating the Reddit API was challenging, especially handling asynchronous data fetching and managing state efficiently. However, as the project progressed, implementing Redux Thunk for middleware logic and structuring the Redux store helped streamline API requests, making data management more predictable and scalable. This approach not only improved the app‚Äôs performance but also reinforced best practices in state management.

Mixtape Tunes

Mixtape Tunes is a React app that uses the Spotify API to search for songs or artists, play songs and add them to a playlist. Users can rename their playlist, search for recommendations based on the songs in their playlist and save the playlist to their Spotify account. The styling of the app is designed in a retro 80s look reminiscent of the era of mixtapes and Walkmans. Accordingly, the colour palette is also kept in the typical bright colours of the 80s. For the front end part I mainly used tailwind and SASS.

Mixtape Tunes is currently under review by Spotify. Until then the login is restricted to users on the whitelist. If you are interested in trying out the app, please send me the email address of your Spotify account and I will add it to the whitelist.

Movienights

Movienights was the final project of my web development bootcamp at Le Wagon Berlin. In a team of 5 people we realized this project from the wireframe to the first Figma prototype to the realization in Ruby on Rails. We implemented different elements with Javascript through Stimulus in Rails, used TMDB as API and PostgresSQL as database. The site includes a dynamic chat, a search function for movies, and a map for mapping movie theater addresses.

The idea behind the project was to give enthusiastic moviegoers the opportunity to connect with each other and arrange to go to the movies together. The app offers both recommended users and films based on your own favorites for films, genres and directors. Other users can participate and chat with each other on their own event page, which is created by users. Even though it was a final project, we are still developing the app.

During the development of the app, I am mainly responsible for the front end part. This includes the implementation of the Figma design with HTML and CSS, as well as the integration of functions with Javascript, and the query of APIs such as mapbox and The Movie Data Base. The project has offered and continues to offer various challenges, which has led to a great learning experience.
