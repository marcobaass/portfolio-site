Movie Nights was the final project of my web development bootcamp at Le Wagon Berlin. In a team of 5 people we realized this project from the wireframe to the first Figma prototype to the realization in Ruby on Rails. We implemented different elements with Javascript through Stimulus in Rails, used TMDB as API and PostgresSQL as database. The site includes a dynamic chat, a search function for movies, and a map for mapping movie theater addresses.

The idea behind the project was to give enthusiastic moviegoers the opportunity to connect with each other and arrange to go to the movies together. The app offers both recommended users and films based on your own favorites for films, genres and directors. Other users can participate and chat with each other on their own event page, which is created by users. Even though it was a final project, we are still developing the app.

During the development of the app, I am mainly responsible for the front end part. This includes the implementation of the Figma design with HTML and CSS, as well as the integration of functions with Javascript, and the query of APIs such as mapbox and The Movie Data Base. The project has offered and continues to offer various challenges, which has led to a great learning experience.
